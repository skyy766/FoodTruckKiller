# 餐车杀手 Food Truck Killer — Bug 追踪模板

> 文档版本：v1.0
> 维护：QA/测试工程师 A6
> 用途：记录、追踪、验证开发与测试过程中发现的缺陷

---

## 字段说明

| 字段 | 说明 |
|------|------|
| BugID | 唯一标识，格式 `BUG-YYYY-NNN`（如 BUG-2026-001） |
| 标题 | 一句话概括问题 |
| 描述 | 详细说明缺陷现象 |
| 复现步骤 | 编号步骤，可重复触发 |
| 预期 | 按设计应发生的行为 |
| 实际 | 实际发生的错误行为 |
| 严重度 | 致命 / 严重 / 一般 / 轻微 |
| 状态 | 新建 / 处理中 / 已修复 / 已验证 |
| 负责人 | 当前处理人（A1 主程 / A2 玩法 / A3 关卡 / A4 美术 / A5 音效） |

### 严重度定义

| 等级 | 定义 | 示例 |
|------|------|------|
| 致命 | 崩溃、数据丢失、完全无法游玩 | 游戏启动崩溃、存档丢失、关卡无法加载 |
| 严重 | 核心功能不可用、流程阻塞 | 击杀无效、出餐不付款、通缉不触发 |
| 一般 | 功能异常但有 workaround | 动画错位、数值偏差、UI 显示错误 |
| 轻微 | 体验问题、不影响功能 | 文案错别字、音效延迟、像素边缘锯齿 |

### 状态流转

```
新建 → 处理中 → 已修复 → 已验证
                              ↓
                         重新打开（验证失败）→ 处理中
```
